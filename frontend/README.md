## Obbiettivo
  Una web app che sia in grado di fornire informazioni sui pokémon e, tramite un chatbot, consigli sul competitivo.

## Stato attuale
  Fornire una lista virtualizzata di pokémon.
  (una lista virtualizzata è un lista che mostra solo un quantitativo predefinito di pokemon, questo per evitare che ci siano troppi elementi nella UI)

## Prossimo step
  Trasformare gli sprite/img in blob e salvare le informazioni in modo da contattare l'api solo su scelta dell'utente oppure se è passato X tempo dall'ultimo check.
  
## Perché sto creando questa app?
Sto costruendo PokeApp per esercitarmi sulle tecnologie che sto imparando mentre cerco lavoro:
- Virtualizzazione liste (react-window), caching Blob/IndexedDB per performance.
- Stack full: React/Vite frontend + Python/FastAPI backend.
So che alcune feature potrebbero essere overkill per un pokedex, ma le uso per pratica e imparare ogni giorno qualcosa di nuovo.
